# 個人化網站專案說明

這是一個使用原生 HTML、CSS、JavaScript 製作的單頁式個人網站原型，主打玻璃擬態導覽、圓形主內容區，以及分類式頁面切換。

## 目前網站結構

網站目前共有 5 個大分類：

- `HOME`
- `ME`
- `WORK`
- `GAME`
- `LINK`

其中 `GAME` 是新增給獨立遊戲專案使用的分類。

## 為什麼 GAME 先分 4 個子分類

我先維持每個大分類 4 個子頁，原因是：

- 和原本網站的切換邏輯一致
- 版型排列最整齊
- 不需要額外重寫更多導覽規則
- 之後擴充內容時也比較好管理

`GAME` 目前先拆成這 4 個子分類：

- `Overview`
- `Story`
- `Gameplay`
- `Devlog`

這樣的拆法很適合獨立遊戲頁面：

- `Overview` 放專案介紹、開發狀態、平台資訊
- `Story` 放世界觀、角色、背景設定
- `Gameplay` 放玩法、系統、特色機制
- `Devlog` 放開發紀錄、更新日誌、製作過程

## 目前功能

- 右上角主分類按鈕可切換網站大分類
- 浮動子選單會依照主分類顯示對應子項目
- 中央圓形內容區會切換對應頁面
- 子選單底部提供上一頁與下一頁切換
- `LINK` 分類中的外部連結會以新分頁開啟
- 手機版會自動調整主選單與子選單位置

## 目前內容狀態

為了先完成版型配置，頁面內容目前先統一使用 `Coming Soon`。

目前總共有：

- 5 個大分類
- 每個分類 4 個子頁
- 共 20 個內容區塊

## 主要技術

- HTML：頁面結構與內容區塊
- CSS：玻璃擬態、動畫、響應式版型
- JavaScript：分類切換、子選單更新、頁面跳轉控制

## 檔案結構

```text
個人化網站/
├─ index.html
└─ README.md
```

## 使用方式

1. 直接以瀏覽器開啟 `index.html`
2. 點擊主分類按鈕
3. 從子選單切換頁面
4. 點擊 `LINK` 內的外部連結即可另開新分頁

## 後續建議

- 把 `GAME` 的 4 個子分類逐步補成正式內容
- 加入遊戲截圖、影片或開發日誌
- 若未來遊戲變多，再把 `GAME` 改成 `GAMES`
- 若之後每個分類頁數不同，可以再把目前固定 4 頁的結構改成完全動態版本
